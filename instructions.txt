Le but du jeu est d'attribuer le code APE correspondant à partir du libellé d'activité, et éventuellement selon les cas, des variables annexes.

Si vous avez des difficultés, passez en cliquant sur "Skip".
Un champ est disponible pour un bref commentaire en cas de besoin.

Si vous détectez une activité d'intermédiation, n'hésitez pas à naviguer directement dans la nomenclature via le moteur de recherche dans la NAF en choisissant "Non". 

L'objectif est d'en expertiser un maximum.

Amusez vous bien !
